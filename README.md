# Tchê Agro - Mobile (Expo)

Versão inicial em React Native (Expo) baseada no layout e telas fornecidos.

Como rodar (Windows PowerShell):

```powershell
cd c:\Users\e_vsjesus\Desktop\devapp
npm install
npm start
# Ou usar: npm run android / npm run ios (se tiver ambiente configurado)
```

O projeto usa uma API mock (`src/api/mock.js`) para simular `Produtor`, `Visita` e `CadernoCampo`.
